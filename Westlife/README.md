# Westlife WordPress Theme

A powerful, elegant and fast WordPress theme for bloggers, creators and developers.

一款优雅、强大且极速的 WordPress 主题，专为博客、创作者与极简内容网站设计。

---

## ✨ Features / 主题特色

- 🌙 Dark/Light Mode auto switch  
  黑暗 / 浅色模式自动切换

- 🚀 Ultra fast loading with CDN static files and Gravatar acceleration  
  极速加载，CDN 静态资源 + Gravatar 全球加速

- 💬 Enhanced comments (visitor IP, browser, OS info, iMessage style layout)  
  评论增强（访客 IP、浏览器、系统信息展示，iMessage 风格布局）

- 📅 Github-like heatmap for posts and talks  
  文章与说说活跃热力图（GitHub 风格）

- 🗺️ Visitor map integration (support 51.la and other statistical tools)  
  访客地图（支持 51.la 及其它统计平台集成）

- 🖼️ Card-based post layout with lazy loading and pagination  
  卡片式文章布局，支持懒加载与分页

- 🔗 Full FontAwesome Pro icon support  
  全局 FontAwesome Pro 图标支持

- 📊 SEO friendly meta and beautiful typography  
  SEO 友好，优雅排版，极致阅读体验

- ⚡️ PJAX ready for fast navigation (optional)  
  支持 PJAX 无刷新体验（可选功能）

- 🧩 Easy to customize and extend  
  易于二次开发与扩展
