## 极速部署指南

1. **环境要求**：
   - PHP >= 7.4 (推荐 PHP 8.1+ 以获得满血性能)
   - MySQL/MariaDB 数据库
   - 必须开启 `pdo_mysql` 和 `mbstring` 扩展
2. **安装步骤**：
   - 将本压缩包内所有文件解压并上传至网站根目录。
   - 浏览器访问 `https://你的域名/install.php`，跟随探针引导完成初始化。
   - **安全警告**：安装成功后，系统会生成 `config.php`，请务必第一时间在服务器上**删除 `install.php` 文件**！
3. **日常使用**：
   - 后台入口：`https://你的域名/admin/login`
